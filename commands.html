Commands

  • /help - Lists all available commands.
  • /add_word - Add a word with its response.
  • /list_words - List all words and responses for this server.
  • /remove_word - Remove a word by its number.
  • /п«пи - Sends a GIF.
  • /store_gif - Store a new GIF URL.
  • /unstore_gif - Remove a stored GIF URL.
  • /warn - Warn a member with an optional reason. (Admin only)
  • /show_warnings - Show warnings for all members or a specific member. (Admin only)
  • /clear_all_warnings - Remove all warnings for all members in the server. (Admin only)
  • /remove_all_warnings - Remove all warnings for a specific member. (Admin only)
  • /remove_warning - Remove a specific warning from a member. (Admin only)
  • /setgif - Set a GIF to be sent after any user message in the current channel.
  • /removesetgif - Remove a set GIF for the current channel.
  • /tax - Calculate Probot tax.
  • /setrole - Set a role to restrict access to a channel.
  • /removesetrole - Remove the role restriction from a channel.
  • /role_permissions - Shows the permissions of a role.
  • /user - Shows information about a user.
  • /joined - Check when a user joined Discord and the server.
  • /send - Send a message with embed by the bot.
  • /change_server_name - Changes the server name.
  • /make_channel - Creates a new channel with the specified name.
  • /make_category - Creates a new category with the specified name.
  • /move_channel - Move a channel to a different category.
  • /ticket - Create a support ticket.
  • /set_welcome - Set a welcome message for new members.
  • /verify - Verify a user.
  • /giveaway - Start a giveaway.
  • /call - Sends a private message to the mentioned user.
  • /serverinfo - Get information about the server.
  • /about - Information about the bot.
  • /kick - Kick a user from the server.
  • /bans - Get a list of members who are banned from this server.
  • /unban - Unban a user from the server.
  • /ban - Ban a user from the server.
  • /timeout - Timeout a user.
  • /untimeout - Remove timeout from a user.
  • /give_role - Give a role to a member.
  • /createrole - Create a new role.
  • /remove_role - Remove a role from a member.
  • /setname - Set a designated name for a user.
  • /slowmode - Enable slow mode in a channel.
  • /hide - Hide a channel.
  • /invite - Get the bot's invite link.
  • /roles - List all roles in the server under categories.
  • /unhide - Unhide a channel.
  • /members - Get the number of members in the server.
  • /clear - Delete a specified number of messages.
  • /lock - Lock a channel to prevent users from sending messages.
  • /unlock - Unlock a channel to allow users to send messages.
  • /mute - Mute a user in the server.
  • /unmute - Unmute a user in the server.
  • /avatar - Get the avatar of a user.
  • /antilink - Disable any user from sending links.
  • /afk - Set yourself as AFK with an optional message. Use /afk all to set yourself as AFK in all servers.
  • /ping - Get the bot's current latency.
  • /give_role_to_all - Give a specified role to all members in the server.
